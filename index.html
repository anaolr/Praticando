QUESTIONÁRIO


QUESTÃO 01:
Explique qual é a função de um servidor web em um ambiente de hospedagem e cite pelo menos um exemplo.

RESPOSTA:
Um servidor web é como um garçom da internet: ele entrega os sites para as pessoas. Quando você digita o endereço de um site (como google.com), esse servidor pega os arquivos do site e mostra no seu navegador. Exemplo: O Apache é um dos servidores web mais famosos. Ele envia páginas da internet para quem acessa.
QUESTÃO 02:
Qual a principal característica do NGINX e por que ele é muito utilizado em projetos de alto desempenho?

RESPOSTA:
O NGINX é rápido e leve. Ele consegue atender muitos acessos ao mesmo tempo sem travar, por isso é ótimo para sites com muito movimento.
QUESTÃO 03:
Diferencie os protocolos HTTP e HTTPS. Por que o HTTPS é recomendado para aplicações modernas?

RESPOSTA:
HTTP não é seguro, qualquer pessoa pode ver os dados. HTTPS é seguro porque os dados são protegidos. Por isso, HTTPS é o ideal para sites com senhas, pagamentos e informações pessoais.
QUESTÃO 04:
Qual é a função das portas em uma rede de computadores e qual a importância da porta 443?

RESPOSTA:
As portas são canais por onde os dados passam. A porta 443 é usada por sites seguros (HTTPS), protegendo as informações dos usuários.
QUESTÃO 05:
Defina o conceito de endereço IP e explique sua importância em redes de computadores.

RESPOSTA:
O IP é o endereço do computador na internet. Ele permite que os dados cheguem ao destino certo, como uma carta com endereço.
QUESTÃO 06:
Explique o que é hospedagem VPS e em quais situações ela é mais indicada.

RESPOSTA:
VPS é como um pedaço reservado de um servidor só para você. É indicada quando o site tem muitos acessos ou precisa de mais segurança e controle.
QUESTÃO 07:
Descreva as vantagens do servidor web Caddy em relação a outros servidores tradicionais.

RESPOSTA:
O Caddy é fácil de usar e já vem com segurança HTTPS automática. É ótimo para quem quer algo simples e seguro.
QUESTÃO 08:
Qual servidor de aplicação é mais indicado para aplicações desenvolvidas na plataforma .NET (C#)? Explique o motivo.

RESPOSTA:
O Microsoft IIS é o mais indicado porque foi feito pela mesma empresa da plataforma .NET, funcionando perfeitamente com ela.
QUESTÃO 09:
Descreva o funcionamento do modelo Cliente-Servidor no contexto de sistemas web.

RESPOSTA:
O cliente é quem faz o pedido (como você no navegador) e o servidor é quem entrega o site. Eles trocam informações para o site funcionar.
QUESTÃO 10:
Explique o que é o Apache Tomcat e para qual tipo de aplicação ele é mais indicado.

RESPOSTA:
O Apache Tomcat é um servidor para rodar sistemas criados em Java. Ele entende essa linguagem e faz os programas funcionarem.
QUESTÃO 11:
Diferencie, com suas palavras, um servidor web de um servidor de aplicação, destacando suas funções principais.

RESPOSTA:
O servidor web mostra páginas simples (como textos e imagens). O servidor de aplicação roda programas mais complexos, como logins e sistemas de compra.
QUESTÃO 12:
Defina o conceito de domínio na internet e sua relação com os endereços IP.

RESPOSTA:
O domínio é o nome do site (como google.com). Ele substitui o IP, que é difícil de memorizar. É como transformar "números" em nomes fáceis.
QUESTÃO 13:
Quais as principais vantagens da hospedagem em nuvem em comparação com a hospedagem local?

RESPOSTA:
A nuvem é mais segura, está sempre online e pode crescer com o seu site. A local depende de você manter tudo funcionando.
QUESTÃO 14:
Cite dois exemplos de servidores de aplicação e as linguagens/plataformas que eles suportam.

RESPOSTA:
Tomcat (Java), Gunicorn (Python). Cada um serve para uma linguagem diferente.
QUESTÃO 15:
Por que é essencial a utilização de HTTPS em sites e sistemas atualmente? Quais riscos o HTTP apresenta?

RESPOSTA:
O HTTPS protege os dados. O HTTP não protege, então alguém pode roubar senhas ou informações pessoais.
QUESTÃO 16:
Dê um exemplo de situação onde o uso de hospedagem VPS seria mais vantajoso do que uma hospedagem compartilhada.

RESPOSTA:
Um site de loja online com muitos acessos precisa de VPS para ter mais segurança e velocidade. A hospedagem compartilhada pode travar.
QUESTÃO 17:
Diferencie os conceitos de IP e porta em uma rede, explicando suas funções.

RESPOSTA:
IP é o endereço do computador. Porta é o canal por onde os dados entram e saem. Juntos, eles fazem a comunicação funcionar.
QUESTÃO 18:
Qual a importância de um servidor web conseguir lidar com conexões via HTTPS?

RESPOSTA:
Porque assim ele garante segurança. Os dados dos usuários ficam protegidos contra roubo.
QUESTÃO 19:
Cite dois exemplos de servidores web e explique suas principais características.

RESPOSTA:
Apache: muito usado, estável. NGINX: rápido, ideal para sites com muitos visitantes.
QUESTÃO 20:
Defina o que é uma hospedagem compartilhada e em quais casos ela costuma ser utilizada.

RESPOSTA:
É quando vários sites dividem o mesmo servidor. Boa para sites pequenos, blogs e projetos pessoais.
QUESTÃO 21:
Explique a função de um servidor de aplicação e a diferença em relação a servidores que apenas servem arquivos estáticos.

RESPOSTA:
O servidor de aplicação roda programas que precisam de lógica. Já o servidor web só mostra arquivos prontos (como textos e imagens).
QUESTÃO 22:
Cite um servidor de aplicação usado para projetos em Python e descreva como ele funciona.

RESPOSTA:
Gunicorn é um servidor que roda sistemas feitos em Python, geralmente junto com frameworks como Flask ou Django.
QUESTÃO 23:
O que é o WildFly (ex-JBoss) e para qual tecnologia ele é indicado?

RESPOSTA:
É um servidor para sistemas feitos em Java, usado em empresas que precisam de sistemas complexos.
QUESTÃO 24:
Qual a função da porta 21 em redes de computadores e para qual serviço ela é utilizada?

RESPOSTA:
A porta 21 é usada para o serviço de FTP, que serve para enviar arquivos para o servidor.
QUESTÃO 25:
Explique por que o servidor LiteSpeed é compatível com o Apache e quais as vantagens dessa compatibilidade.

RESPOSTA:
Porque ele entende os mesmos arquivos e configurações do Apache. Isso facilita a troca de servidor e melhora o desempenho.
QUESTÃO 26:
O que é o Microsoft IIS e em quais situações ele costuma ser adotado?

RESPOSTA:
É o servidor da Microsoft. Usado quando os sites são feitos em tecnologias como C# e .NET.
QUESTÃO 27:
Explique para que serve o Node.js e em quais tipos de aplicações ele é mais utilizado.

RESPOSTA:
Node.js é usado para criar sites e sistemas rápidos com JavaScript. É ideal para chats, jogos e sistemas em tempo real.
QUESTÃO 28:
Descreva a função do servidor GlassFish e para qual linguagem ele é destinado.

RESPOSTA:
O GlassFish roda sistemas feitos em Java. É usado em aplicações corporativas.
QUESTÃO 29:
O que são Puma e Unicorn e em quais projetos eles são normalmente utilizados?

RESPOSTA:
São servidores usados em sites feitos com Ruby (Ruby on Rails). Eles ajudam a deixar o site mais rápido.
QUESTÃO 30:
Qual a importância da escolha adequada entre hospedagem local, compartilhada, VPS e nuvem no planejamento de um projeto web?

RESPOSTA:
Porque cada tipo serve para uma situação. A escolha errada pode deixar o site lento, fora do ar ou inseguro. Planejar isso ajuda o projeto a funcionar bem.
